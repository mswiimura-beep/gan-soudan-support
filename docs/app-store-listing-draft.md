# App Store Listing Draft

## App Name

がん相談サポート

## Subtitle

がんの不安と質問を整理

## Promotional Text

診察前や相談前に、不安、聞きたいこと、伝えたいことを整理するための支援アプリです。

## Description

がん相談サポートは、がん患者さん、家族、付き添い者が、診察前や相談前に不安や疑問を整理するためのアプリです。

相談メモを作成すると、主治医に聞く質問、がん相談支援センターに相談できること、診察前に見返すチェック項目を整理できます。PDFまたはテキストとして出力し、診察時や相談時に見せるメモとして使えます。

このアプリは、診断、治療方針、薬剤、緊急性の判断を行いません。強い症状や危険を感じる場合は、アプリ内で判断せず、医療機関や救急に連絡してください。

主な機能:

- 相談メモの作成、編集、検索、削除
- 患者本人、家族、付き添い者などの立場の整理
- 主治医に聞く質問リスト
- がん相談支援センターに相談できることの整理
- 診察前チェックリスト
- 信頼できる情報へのリンク
- PDF・テキスト出力
- 端末内保存と相談履歴削除
