# Manual QA Checklist

Run this on a real device or a healthy Simulator before TestFlight.

## First Launch

- 初回起動で同意画面が出る
- 2つの確認チェックを入れるまで開始ボタンが押せない
- 同意後にホームへ移動する

## Consultation Notes

- よくある相談テンプレートをタップするとカテゴリ、立場、伝える相手、本文が入る
- 本文が空の時は保存ボタンが無効
- 相談メモを保存できる
- 保存後にホームの最近の相談メモへ表示される
- 相談予定日を入れるとホームの「次の診察・相談予定」に表示される
- メモ詳細から編集できる
- メモ詳細から1件削除できる
- 設定から全件削除できる

## Search And Export

- 相談メモ一覧で本文、カテゴリ、立場、伝える相手、予定日で検索できる
- 全件PDFを作成して共有シートが出る
- 全件テキストを作成して共有シートが出る
- 全件の内容をコピーできる
- 1件PDFを作成して共有シートが出る
- 1件テキストを作成して共有シートが出る
- 1件の内容をコピーできる
- 短く伝える文だけをコピーできる

## Safety

- 「死にたい」「消えたい」などを含むメモで赤い安全案内が最上部に出る
- 「胸痛」「呼吸困難」「強い痛み」などを含むメモで緊急時案内が出る
- ホーム、設定、アプリの位置づけから緊急時の案内画面へ移動できる
- 緊急時の案内画面で119、医療機関、身近な人、自傷時の連絡が案内される
- 治療法、薬の中止、再発、余命を断定する文が出ない

## Privacy

- 設定からプライバシー要点を読める
- 設定からプライバシーポリシー案を読める
- 設定からアプリの位置づけを読める
- 相談内容が外部送信されない前提の文言になっている
