# 立場別 疑似相談テスト検討会メモ

実施日: 2026年7月9日

## 実施内容

- 本人、家族、友人、同僚の4立場で疑似相談を実施
- 各立場50パターン、合計200パターン
- テーマは副作用、治療説明、医療費、仕事、家族、見た目の変化、セカンドオピニオン、緩和ケア、気持ちのつらさ、緊急・自傷表現など
- 確認項目は、分類、質問案、短く伝える文、安全表示、禁止表現の有無

## 検討会での確認

- 本人・家族・友人・同僚のいずれでも、診断や治療判断に踏み込む禁止表現は検出されなかった。
- 友人・同僚はアプリ上では「支える人」に寄せて処理され、本人の意思決定を代行しない文脈が保たれた。
- 緊急・自傷表現は全ロールで critical として扱われ、相談整理より連絡優先の文が出た。
- 家族・友人・同僚の相談では、『本人に確認する範囲』と『本人の同意なく共有しない情報』を分けて表示できた。
- 同僚ケースでは、職場復帰や配慮の話と、診断名や治療内容を詳しく伝えなくてよい範囲を分けて表示できた。

## サンプル出力

### 本人 / 病状説明後の不安

- 分類: 治療選択、伝え方・聞き方、家族・人間関係
- 伝える文: 今日は「本人の立場です。先生から病状説明を受けましたが、家に帰ると不安だけが残り、何を聞けばよいか分からなくなりました。」について相談したいです。特に、今いちばん整理したいのは、治療、症状、お金、家族、仕事、気持ちのどれに近いですか？ また、どの状態なら病院へ連絡すべきか確認したいです。
- 主治医に聞くこと: 説明で分からなくなった点を、もう一度聞き直してもよいですか？
- 相談員・相談窓口に聞くこと: 相談内容を整理するために、どの情報を持参するとよいですか？
- 共有前の確認: なし
- 安全表示: なし

### 本人 / 医療費と収入

- 分類: 医療費・制度、治療選択、仕事・学校
- 伝える文: がんの治療や生活のことで「本人の立場です。治療費、収入減、制度の手続きが心配です。どこに何を相談すればよいか分かりません。」が整理できずにいます。医師に聞くこと、使える制度や相談先、家族や職場への伝え方を一緒に整理したいです。
- 主治医に聞くこと: 治療予定、通院頻度、仕事や収入に影響しそうな時期を確認できますか？
- 相談員・相談窓口に聞くこと: 利用できる医療費支援制度はありますか？
- 共有前の確認: 職場・学校に伝えること: 必要な配慮、休み方、連絡方法、復帰時に困りそうなこと。

## 次の改善候補

- 友人・同僚向けの表示を、さらに短い言葉にして画面負担を下げる。
- 家族向けには、家族自身の不安と本人の意思尊重を同時に扱える入力例を増やす。
- 公式情報の前に読む短い説明を増やし、難しい公的サイトに飛ぶ前の心理的負担を下げる。
