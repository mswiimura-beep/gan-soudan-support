# 公開テスト配布ガイド

目的は、インターネットにつながる人ならURLだけで試せる状態にし、感想を集めて改善することです。

知り合い以外にも広く配布するオープンベータでは、[open-beta-guide.md](./open-beta-guide.md) も確認してください。

## 推奨構成

- 公開先: GitHub Pages、Netlify、Vercelなどの静的ホスティング
- アプリ本体: `web/` フォルダ
- 感想回収: Googleフォームなどの外部フォーム
- 個人情報: テスターには個人名、病院名、主治医名、患者番号、連絡先、詳しい診断情報、相談メモ本文を書かないよう案内する

## テスト段階の個人情報ルール

- 実名、病院名、主治医名、患者番号、住所、電話番号、メールアドレスは入力しない
- 詳しい診断名、病期、治療歴、検査値など、本人が特定され得る医療情報は入力しない
- 感想フォームには相談メモ本文を貼り付けない
- Googleフォーム等ではメールアドレス収集をオフにする
- Googleフォーム等ではログイン必須にしない
- 回答一覧を閲覧できる人を最小限にする
- 公開テスト中に集めるのは、使いやすさ、迷った場所、分かりにくい言葉、改善希望に限定する
- 誤って個人情報を受け取った場合は、保存せず削除する運用にする

## GitHub Pagesで公開する場合

1. このプロジェクトをGitHubのリポジトリへpushします。
2. GitHubのリポジトリ画面で `Settings > Pages` を開きます。
3. `Build and deployment` を `GitHub Actions` にします。
4. `main` ブランチへpushすると `.github/workflows/pages.yml` が `web/` を公開します。
5. Actions完了後に表示されるURLをテスターへ共有します。

## 感想フォームをつなぐ

Googleフォームなどで、以下の質問を作ります。

- 使った人に近いもの
- 迷ったところ
- 良かったところ
- 改善してほしいところ
- 自由記載

フォーム説明文には、次の文を入れてください。

```text
個人名、病院名、主治医名、患者番号、住所、電話番号、詳しい診断情報、相談メモ本文は入力しないでください。
このフォームでは、使いやすさ・分かりにくかった言葉・迷った操作・改善希望だけを集めます。
```

フォームURLを作成したら、[app.js](./app.js) の先頭付近にある `FEEDBACK_FORM_URL` へ設定します。

```js
const FEEDBACK_FORM_URL = "https://forms.gle/...";
```

設定前でも、Webアプリ内の「感想を送る」から感想テンプレートをコピーできます。

## テスターへ送る文面

以下をそのまま使えます。

```text
がん相談サポート Webの試作版をテストしています。
診断や治療判断をするアプリではなく、相談内容を整理するためのアプリです。

試してほしいこと:
1. 「自分のことから書き出す」から相談メモを1つ作る
2. 「整理」で、モヤモヤの要約と主治医や相談支援センターに聞くことを見る
3. 「情報」で、困りごと別の要点を見る
4. 「感想を送る」から、迷ったところ・良かったところを教えてください

注意:
個人名、病院名、主治医名、患者番号、住所、電話番号、詳しい診断情報、相談メモ本文は入力しないでください。
感想フォームには、使いやすさ・迷ったところ・分かりにくかった言葉だけを書いてください。
強い症状や危険を感じる場合は、アプリではなく医療機関や救急へ連絡してください。
```

## 公開前チェック

```sh
node --check web/app.js
node --check web/sw.js
node web/check-web-app.mjs
node web/mock-consultation-scenarios.mjs
node web/pdca-120-simulations.mjs
python3 -m json.tool web/manifest.webmanifest >/dev/null
```

模擬相談テストでは、以下のような相談で動作を確認します。

- 副作用と生活負担
- 治療選択の迷い
- お金、仕事、休職、職場復帰
- 家族への伝え方
- 気持ちのつらさ
- セカンドオピニオン
- 緩和ケアへの不安
- 説明が分からない、聞き直したい
- がん相談支援センターの使い方
- 若い世代の学校、仕事、将来子どもが欲しい気持ち
- 見た目の変化
- 急な強い症状
- 自傷につながる表現
